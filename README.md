# 🕒 Habit Tracker

A modern, visually appealing **Habit Tracker App** built using **React**, **Vite**, and **Tailwind CSS**.  
This application helps users build strong habits, stay consistent, and track their progress through an intuitive and engaging interface.

---

## 🌐 Live Link  
🔗 https://habit-tracker-9e59c.web.app

---

## 🚀 Features

- ✅ **User Authentication** — Secure login & signup powered by Firebase  
- ✅ **Track Habits** — Add, edit, delete, and monitor habits effortlessly  
- ✅ **Progress Visualization** — View streaks, charts, and progress indicators  
- ✅ **Responsive UI** — Fully optimized for mobile, tablet, and desktop  
- ✅ **Interactive Animations** — Smooth animations using Framer Motion  
- ✅ **Notifications & Alerts** — Toasts & modals with React Hot Toast & SweetAlert2

## 📦 Installation

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/habit-tracker.git
cd habit-tracker
```

## 2️⃣ Install Dependencies
```bash
npm install
```

## 3️⃣ Start Development Server

```bash
npm run dev
```

Then open: http://localhost:5173

## 🔑 Firebase Setup

1. Go to the **Firebase Console**  
2. Create a **new Firebase project**  
3. Navigate to **Authentication → Sign-in Methods** and enable:  
   - **Email/Password Authentication**  
   - **Google Sign-In**  
4. Go to **Project Settings → SDK Setup**, copy your Firebase configuration, and paste it into your **.env** file  
5. Initialize Firebase inside your project (e.g., `firebase.init.js`) and export the auth/app instances for use in your Habit Tracker
